# Contributing to SplitSmart AI

First off, thank you for considering contributing to SplitSmart AI! It's people like you that make SplitSmart AI such a great tool.

## 🌈 How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issue Tracker.
- Describe the bug and include steps to reproduce.
- Include information about your environment (browser, OS).

### Suggesting Enhancements
- Check if the enhancement has already been suggested.
- Open an issue with a clear title and description.
- Explain why this enhancement would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Issue that pull request!

## 📜 Code of Conduct
Please be respectful and kind to others in the community. We follow the standard Contributor Covenant.

## 🛠️ Development Setup
1. Clone your fork.
2. Install dependencies: `npm install`.
3. Start dev server: `npm run dev`.
4. Create a `.env` with your Supabase credentials.
